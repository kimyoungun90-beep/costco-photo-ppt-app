코스트코 진열·연출 점검 PPT 생성기 v10

배포 파일
- index.html
- pptxgen.min.js

주요 변경
- 페이지를 열거나 점포·HUB 촬영 주차를 변경하면 HUB 사진을 자동으로 불러옵니다.
- 'HUB 사진 새로고침' 버튼은 자동 연동이 지연될 때 수동 재확인용입니다.
- 기본 표는 3행 × 6분류를 유지합니다.
- 한 분류의 사진이 3장을 넘으면 가장 아래 추가 사진 행에 자동 배치합니다.
- 비네트 4장, 에어컨 5장 등 총 20개 표준 촬영 사진을 한 장에 배치할 수 있습니다.
- 추가 행에서도 PPT 사진 크기는 4cm × 4cm로 유지합니다.
- 추가 사진에는 항목명이 표시되어 다른 분류 열에 놓여도 내용을 구분할 수 있습니다.

연동 조건
- HUB v129 Code.gs를 Apps Script 새 버전으로 재배포해야 합니다.
- HUB와 진열연출앱의 점포 및 촬영 주차가 같아야 합니다.
- PPT분류는 TV / REF / AC / MON / COOK / ETC를 사용합니다.
